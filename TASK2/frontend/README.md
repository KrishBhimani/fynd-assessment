# Frontend - AI Feedback System

Next.js frontend with shadcn/ui components.

## Setup

```bash
npm install
cp .env.local.example .env.local
npm run dev
```

## Pages

- `/` - User Dashboard (submit reviews)
- `/admin` - Admin Dashboard (view all submissions)

## Environment Variables

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```
